# Sistema de Gerenciamento de Funcionários

Sistema em Java para gerenciar funcionários CLT, PJ e Estagiários.

## Funcionalidades
- Listar funcionários ordenados por salário
- Adicionar novo funcionário
- Calcular folha de pagamento total
- Listar tributáveis com impostos
- Buscar por CPF
- Gerar relatório em .txt

## Tecnologias
- Java 17+
- POO (Herança, Polimorfismo, Interfaces)
- Exceções customizadas
